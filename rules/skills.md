# Skills

## Unskilled vs Skilled
- skilled bonus tied to advancement
- skills tied to ability scores (highest is skilled)
- knowledge skills (set number chosen during character creation)


# Strength Skills 
- Climb/Swim/Jump/Run
- Lift/Drag/Pull/Push


# Agility Skills
- Sneak/Hide
- Acrobatics
- Fine-motor


# Acuity Skills
- Perception (See/Hear/Smell)
- Search
- Insight


# Will Skills
- Persuasion
- Deception
- Intimidation


# Knowledge Skills
- Sorcery 
- Nature
- Delving
- Religion
- History
- Medicine
- Music
- Craft
- Lockpicking
